Overview

Poultry feed quality directly affects bird health and productivity. This project allows farmers or feed manufacturers to quickly assess feed quality through image analysis. It uses the Gemini API to detect visual inconsistencies such as color variation, dust, mold, or broken pellets.

Features

Good Feed / Bad Feed Classification: Provides percentage estimates of good and bad feed.

Visual Quality Analysis: Gives detailed reasoning about the feed quality.

Automated Assessment: Eliminates the need for manual inspection.

Easy to Use: Run a single Python script with an image as input.

Python 3.10+

Gemini API key
